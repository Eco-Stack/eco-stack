### 주제
- OpenStack을 활용해 구축한 **클라우드내 리소스를 모니터링** 하는 웹어플리케이션 입니다.
- 리소스는 컴퓨팅 자원을 의미하며, **CPU / Memory / Disk / Network** 총 4가지 메트릭을 수집합니다.
- 사용자가 **어떤 데이터를 우선적으로 파악하고 싶어하는지**, 사용자에게 **차트를 통해 어떤 인사이트를 제공할 수 있는지**에 대해 고민해볼 수 있었던 프로젝트입니다.
